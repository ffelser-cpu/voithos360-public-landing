# Editing This Landing Page

This is the simple public Voithos360 landing page. It does not give anyone access to the beta app.

## Files

- `index.html` - edit the words and page sections here.
- `styles.css` - edit colors, spacing, and visual style here.
- `script.js` - controls the signup form email draft.
- `headshot.jpg` - replace this file to update the founder photo.
- `demo-video.mp4` - add this file when your short demo video is ready.

## Add Your Video

1. Put your video in this folder.
2. Rename it exactly: `demo-video.mp4`
3. Redeploy/upload the whole folder.

## Change Signup Form

In `script.js`, replace the Google Forms URL assigned to:

`earlyAccessFormUrl`

with the new form URL.

## Deploy

Upload the contents of this folder as a separate public landing page project:

`PUBLIC_LANDING_UPLOAD_THIS`

Keep the beta app project separate and private.
